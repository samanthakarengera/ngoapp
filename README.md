# Enterprise Apps Samantha Karengera

## Over dit project
Dit project is een Spring Boot webapplicatie voor een NGO in Anderlecht.
Dit werkstuk is voor het vak Enterprise Applications.
De NGO helpt mensen een overzicht te geven van evenementen die worden georganiseerd in de buurt voor een betere gemeenschap.

---

## Functionaliteiten

### Homepagina
- Toont een tabel met de 10 laatste evenementen
- Titel en organisatie worden getoond in die tabel
- Link naar details van elk evenement
- Knop om een nieuw evenement toe te voegen

### Nieuw evenement
- Formulier om een event toe te voegen


### Details
- Toont alle informatie van één evenement

### About pagina
- Info over de NGO in Anderlecht

### Contact pagina
- Contactformulier (Mailtrap)

---

## Gebruikte technologieën
- Java
- Spring Boot
- Spring Data JPA
- Thymeleaf
- H2 database (in memory)
- HTML en CSS
